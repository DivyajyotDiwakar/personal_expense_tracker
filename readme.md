# Personal Expense Tracker

A responsive web-based expense management application built using:

- HTML
- CSS
- JavaScript
- jQuery
- Bootstrap 5

## Features

- Add new expenses
- Edit existing expenses
- Delete expenses
- Persistent storage using localStorage
- Responsive UI with Bootstrap 5

## How to Run

1. Clone the repository
2. Open index.html in browser

## Future Improvements

- Category filtering
- Monthly reports
- Charts using Chart.js
- Backend integration
